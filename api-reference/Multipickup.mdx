---
title: "Multipickup"
---
Multipickup Tiers allows bookings to be booked via batches.<br/>
Each batch is picked up from a given location and you can have distinct batches to be picked up from distinct locations.<br/>
Each batch will also be booked on a given date and you will able to complete the booking once you have assigned the minimums shipments amounts for each batch.<br/>
Multipickup bookings gives you access to cheaper Courier rates since this process reduces the operational costs for Couriers, which will be able to pickup multiple shipments from a given location on a given day.
## Set Up
<ul>
<li>You can setup the API to always return the Multipickup Tier price instead of the Everday Price by editing the API module Here.<img src="/images/transdirect-image3.png" /><img src="/images/transdirect-image4.png" /></li>
</ul>
When you enable a Tier Price option, the quotes returned by the API will always be from Tier prices.<br/>
Please note that currently only Couriers Please and Fastway offer Multipickup.

## Confirm a Multipickup booking
This is similar as a normal API confirmation functionality but you will need to specify the tier.
<RequestExample>
```
--header 'Content-Type: application/json'\
--header 'Authorization : Bearer <fec5ae54d239f1bbf23a12010989ba17>'\
--data-raw '{
   {
    "courier": "fastway",
    "pickup-date": "2021-09-17",
    "tier": 1
}
}
``` 
</RequestExample>
<ResponseExample>
```json Response
204
```
</ResponseExample>
