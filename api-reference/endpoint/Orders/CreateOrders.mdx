---
title: "Create Orders"
api: "POST https://www.transdirect.com.au/api/orders"
---
It is required that the body contains the fields by selecting the ‘Create a saved order’ link above. Once an Order has been created, you can retrieve a quote id for a booking by using the PUT method.
<RequestExample>
```bash Example Request
curl --location --request POST 'https://www.transdirect.com.au/api/orders'
--header 'Content-Type:application/json'\
--data-raw '{
{
  "transdirect_order_id": 12345,
  "order_id": "4444",
  "goods_summary": "This is a test",
  "goods_dump": "Another test",

```
